# ⌨️ DevSpace

> The all-in-one collaborative IDE built for student dev teams.

DevSpace eliminates the context-switching problem — no more juggling VS Code, Zoom, Slack, and Jira in separate windows. Everything lives in one native desktop app.

## ✨ Features

- **Live Collaborative Editor** — real-time multiplayer code editing powered by Yjs + Monaco
- **Built-in Voice & Video** — WebRTC calls embedded directly in the workspace, no Zoom needed
- **Domain Workspaces** — frontend and backend developers work in isolated spaces that merge cleanly
- **Contribution Tracking** — see exactly who is doing what, eliminating the one-man-show problem
- **Project Management** — create projects, assign team members, define domains in minutes
- **Chill Zone** — a dedicated social space for the team separate from the work environment

## 🛠 Tech Stack

| Layer | Technology |
|---|---|
| Desktop Shell | Tauri (Rust) |
| Frontend | React + TypeScript + Vite |
| Code Editor | Monaco Editor |
| Real-time Sync | Yjs + Socket.io |
| Voice/Video | WebRTC |
| Backend | Node.js + Express |
| Database | PostgreSQL + Prisma |
| 3D Elements | Three.js + Spline |

## 🚀 Getting Started

```bash
git clone https://github.com/YourUsername/devspace.git
cd devspace
npm install
npm run tauri dev
```
